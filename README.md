# Introduction

This codebase is a sharing & guide session on fundamental data engineering. It consist of 4 session / module. Where each session contain incremental and continuity lesson to build up kniwledge in data engineering. By the end of the forth session, learner would have solid knowledge and a project to showcase for securing entry level job.
